Data Visualization and Statistics Projects

by Brian Delgado

Using the Materials

This project uses real match data to explore the goal-scoring careers of four elite forwards. Each tile below corresponds to one player and includes links to the data, a Tableau, and Excel dashboard, and a brief description of suggested analysis ideas. All following Scientific Report be here.

Goal-Scoring Careers Project

Lionel Messi

  • Data: Excel File and Excel_Dashboard
  • R: R Code
  • Dashboard: Tableau Workbook
  • Project: Aggregate goals by year and model Messi’s scoring output as a function of time. Estimate when he reaches peak scoring form and how quickly his performance declines afterward.
  • Skills: Time-series plots, curve fitting, discrete derivatives, interpreting slopes and maxima.
  • Notes: Interesting to compare his Barcelona years with his later seasons. Consider how league, teammates, and age might influence the trend.

Cristiano Ronaldo

  • Data: Excel File and Excel_Dashboard
  • R: R Code
  • Dashboard: Tableau Workbook
  • Project: Build a yearly goals time series and analyze how his scoring changes as he moves between leagues and teams. Identify his peak seasons using a fitted curve.
  • Skills: Multi-club time-series analysis, polynomial modeling, comparison of growth and decline rates.
  • Notes: Transfers between clubs may create visible jumps in the data. Discuss how changes in role or league strength may affect his curve.

Neymar

  • Data: Excel File and Excel_Dashboard
  • R: R Code
  • Dashboard: Tableau Workbook
  • Project: Examine Neymar’s progression from early career to more recent seasons. Use derivatives to highlight periods of rapid improvement or reduced output.
  • Skills: Smoothing noisy data, discrete rate-of-change calculations, linking numerical patterns to possible injuries or role changes.
  • Notes: Missing seasons or injuries can create gaps; these can be mentioned as limitations in the analysis.

Robert Lewandowski

  • Data: Excel File and Excel_Dashboard
  • R: R Code
  • Dashboard: Tableau Workbook
  • Project: Analyze Lewandowski’s scoring consistency over time and compare his peak years to those of the other forwards in this project.
  • Skills: Comparing multiple fitted curves, analyzing long periods of high performance, visual comparison of different players.
  • Notes: His long prime in top European leagues makes him a good example of sustained peak performance.
